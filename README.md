# 票星球自动抢票

1.安装python3

2.配置config.py

3.安装request
```shell
pip3 install requests
```

4.运行main.py

> 参数1：表示谁，参数2：表示场次（从1开始），参数3：表示票价次序（按照票价从低到高，从1开始）

4.1 Windows
```shell
python .\main.py xzq 1 1
```
4.2 Mac
```shell
  python3 main.py xzq 1 1
```
```shell
  python3 main.py zjl 1 1
```
终止进程
control+c
